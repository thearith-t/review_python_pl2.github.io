PART 1: Review

Chapter 1: Data Types

1.1 — What is a Data Type?

Every value in Python has a type that tells Python what kind of data it is and what operations are allowed.

x = 42          # int
y = 3.14        # float
name = "Alice"  # str
flag = True     # bool
nothing = None  # NoneType

Tip

Use type() to check the type of any value:

print(type(42))       # <class 'int'>
print(type("hello"))  # <class 'str'>
print(type(True))     # <class 'bool'>

1.2 — Numeric Types

int — whole numbers (no decimal):

age = 25
year = 2025
negative = -10

float — numbers with a decimal point:

price = 9.99
temperature = 36.6
pi = 3.14159

Type conversion:

int(3.9)      # → 3  (truncates, does NOT round)
float(5)      # → 5.0
int("42")     # → 42
str(100)      # → "100"

1.3 — Strings

A string is a sequence of characters enclosed in quotes:

s1 = "Hello"
s2 = 'World'
s3 = """Multi
line string"""

Key properties: - Indexed — each character has a position starting at 0 - Immutable — you cannot change individual characters after creation - Iterable — you can loop over each character

s = "Python"
print(s[0])    # P
print(s[-1])   # n
print(s[1:4])  # yth
print(len(s))  # 6

1.4 — Booleans & None

bool — only two values: True or False

is_active = True
is_admin  = False

# These expressions evaluate to bool:
print(10 > 5)       # True
print(10 == 5)      # False
print(type(True))   # <class 'bool'>

None — represents the absence of a value:

result = None
if result is None:
    print("No result yet.")

Note

None is NOT the same as 0, "", or False — but all are falsy.

Chapter 2: Operations

2.1 — Arithmetic Operations

Operator Name Example Result
+ Addition 7 + 3 10
- Subtraction 7 - 3 4
* Multiplication 7 * 3 21
/ Division (float) 7 / 2 3.5
// Floor Division 7 // 2 3
% Modulo 7 % 2 1
** Power 2 ** 8 256 
print(17 // 5)   # 3  (quotient only)
print(17 % 5)    # 2  (remainder only)
print(2 ** 10)   # 1024

2.2 — Comparison & Logical Operators

Comparison — always return bool:

==  !=  >  <  >=  <=
print(5 == 5)    # True
print(5 != 3)    # True
print(10 >= 10)  # True

Logical:

and  # Both must be True
or   # At least one must be True
not  # Reverse the bool
age = 20
print(age >= 18 and age < 65)   # True
print(age < 10 or age > 18)     # True
print(not True)                  # False

2.3 — String Operations

# Concatenation
first = "Hello"
second = "World"
print(first + " " + second)    # Hello World

# Repetition
print("Ha" * 3)                # HaHaHa

# Membership
print("ell" in "Hello")        # True
print("xyz" not in "Hello")    # True

# Length
print(len("Python"))           # 6

Slicing:

s = "Programming"
print(s[0:4])     # Prog
print(s[::2])     # Pormig  (every 2nd char)
print(s[::-1])    # gnimmargorP  (reversed)

Chapter 3: Printing & Formatting

3.1 — The print() Function

print("Hello")                    # Hello
print("a", "b", "c")             # a b c
print("a", "b", sep="-")         # a-b-c
print("Hello", end="!")           # Hello! (no newline)
print()                           # blank line

Printing variables:

name = "Sothearith"
score = 95
print("Name:", name)
print("Score:", score)

3.2 — f-Strings (Formatted String Literals)

The modern, recommended way to format strings in Python 3.6+:

name = "Alice"
score = 87.5

print(f"Student: {name}")
print(f"Score: {score:.2f}")       # 87.50
print(f"Score: {score:.0f}")       # 88
print(f"Value: {10 * 3}")          # Value: 30

Alignment & padding:

print(f"{'Name':<15} {'Score':>6}")
print(f"{'Alice':<15} {87.5:>6.1f}")
# Name             Score
# Alice             87.5

3.3 — Other Formatting Methods

.format() method:

print("Hello, {}!".format("World"))
print("{0} + {1} = {2}".format(3, 4, 7))

% operator (older style):

print("Name: %s, Age: %d" % ("Bob", 30))
print("Pi = %.4f" % 3.14159)

Escape sequences:

print("Line1\nLine2")   # newline
print("Tab\there")      # tab
print("She said \"Hi\"")  # quotes inside string

Chapter 4: Lists

4.1 — What is a List?

A list is an ordered, mutable (changeable) collection that can hold items of any type:

fruits = ["apple", "banana", "cherry"]
numbers = [10, 20, 30, 40]
mixed  = [1, "hello", True, 3.14]
empty  = []

Indexing & Slicing:

fruits = ["apple", "banana", "cherry", "date"]
print(fruits[0])      # apple
print(fruits[-1])     # date
print(fruits[1:3])    # ['banana', 'cherry']
print(fruits[::-1])   # reversed list

4.2 — List Methods

Method What it does Example
.append(x) Add x to end lst.append(5)
.insert(i, x) Insert x at index i lst.insert(0, 1)
.remove(x) Remove first x lst.remove("a")
.pop(i) Remove & return item at i lst.pop()
.sort() Sort in place lst.sort()
.reverse() Reverse in place lst.reverse()
.index(x) Find index of x lst.index(3)
.count(x) Count occurrences of x lst.count(2)
.clear() Remove all items lst.clear()
.copy() Shallow copy lst2 = lst.copy()
len(lst) Number of items len(lst)

4.3 — List Methods in Action

nums = [3, 1, 4, 1, 5, 9, 2, 6]

nums.append(7)
print(nums)            # [3, 1, 4, 1, 5, 9, 2, 6, 7]

nums.remove(1)         # removes first 1
print(nums)            # [3, 4, 1, 5, 9, 2, 6, 7]

popped = nums.pop()    # removes last item
print(popped)          # 7

nums.sort()
print(nums)            # [1, 2, 3, 4, 5, 6, 9]

print(nums.count(1))   # 1
print(nums.index(5))   # 4

Chapter 5: Tuples

5.1 — What is a Tuple?

A tuple is an ordered, immutable (unchangeable) collection:

point   = (3, 7)
colors  = ("red", "green", "blue")
single  = (42,)       # ← comma required for single item!
empty   = ()
no_paren = 1, 2, 3    # parentheses optional

Why use tuples instead of lists?

  • Data that should NOT change (coordinates, RGB values, dates)
  • Slightly faster than lists
  • Can be used as dictionary keys (lists cannot)
point = (3, 7)
# point[0] = 10   ← TypeError! Tuples are immutable
print(point[0])   # 3 ✅ reading is fine

5.2 — Tuple Operations

t = (10, 20, 30, 20, 40)

# Indexing & slicing (same as list)
print(t[1])         # 20
print(t[-1])        # 40
print(t[1:3])       # (20, 30)

# Methods (only 2!)
print(t.count(20))  # 2
print(t.index(30))  # 2

# Length
print(len(t))       # 5

# Unpacking
x, y = (3, 7)
print(x, y)         # 3 7

a, *rest = (1, 2, 3, 4)
print(a, rest)      # 1 [2, 3, 4]

Chapter 6: Dictionaries

6.1 — What is a Dictionary?

A dictionary stores data as key: value pairs. Keys are unique and immutable; values can be anything:

student = {
    "name": "Alice",
    "age": 20,
    "gpa": 3.8
}

print(student["name"])    # Alice
print(student["gpa"])     # 3.8

Accessing safely with .get():

print(student.get("age"))          # 20
print(student.get("score", 0))     # 0 (default if key missing)

6.2 — Dictionary Methods

Method What it does
d[key] = val Add or update
del d[key] Delete a key
d.get(k, default) Safe access
d.keys() All keys
d.values() All values
d.items() All (key, value) pairs
d.update({...}) Merge another dict
d.pop(key) Remove and return value
key in d Check membership
len(d) Number of pairs 
d = {"a": 1, "b": 2}
d["c"] = 3
print(list(d.keys()))    # ['a', 'b', 'c']
print(list(d.values()))  # [1, 2, 3]
for k, v in d.items():
    print(k, "→", v)

Chapter 7: String Methods

7.1 — Essential String Methods

s = "  Hello, World!  "

s.strip()           # "Hello, World!"   (remove whitespace)
s.lower()           # "  hello, world!  "
s.upper()           # "  HELLO, WORLD!  "
s.replace("World", "Python")   # "  Hello, Python!  "
s.split(", ")       # ['  Hello', 'World!  ']
", ".join(["a","b","c"])       # "a, b, c"

7.2 — More String Methods

s = "Hello, Python!"

s.startswith("Hello")   # True
s.endswith("!")         # True
s.find("Python")        # 7  (index where it starts)
s.count("l")            # 2
s.isdigit()             # False
s.isalpha()             # False (has comma & space)
s.capitalize()          # "Hello, python!"
s.title()               # "Hello, Python!"
s.center(20, "*")       # "***Hello, Python!***"
s.zfill(20)             # left-pads with zeros

Important

String methods return new strings — they do NOT modify the original (strings are immutable)!
